# 여운 비주얼 에셋 가이드

## 목적

이 폴더는 여운 앱에 들어갈 시각 에셋을 일관되게 만들기 위한 운영 문서다.

- 어떤 아이콘이 필요한지 정리한다.
- 화면마다 어떤 사진 슬롯이 필요한지 정리한다.
- ChatGPT 이미지 생성으로 바로 뽑을 수 있는 공통 프롬프트와 상세 프롬프트를 제공한다.

## 파일 구성

- `gamdojang_asset_map.md`
  - 아이콘 인벤토리
  - 화면별 사진 슬롯
  - 권장 비율 / 생성 해상도 / 파일 네이밍
- `gamdojang_chatgpt_image_prompts.md`
  - 공통 브랜드 프롬프트
  - 사진용 / 일러스트용 공통 프롬프트
  - 네거티브 제약
  - 에셋별 상세 생성 프롬프트

## 권장 작업 순서

1. `gamdojang_asset_map.md`에서 어떤 에셋이 필요한지 먼저 본다.
2. `gamdojang_chatgpt_image_prompts.md`에서 공통 프롬프트를 읽는다.
3. 먼저 `스타일 앵커` 이미지를 1~2장 생성한다.
4. 이후 피드 사진, 찰나 사진, 온보딩/빈 상태 일러스트를 같은 공통 프롬프트로 개별 생성한다.
5. 같은 시리즈는 가급적 같은 세션에서 생성하고, 가능하면 직전에 생성한 이미지를 레퍼런스로 다시 넣어 통일감을 유지한다.

## 중요한 운영 원칙

- 아이콘은 가능하면 AI 이미지 생성보다 SVG 또는 직접 제작을 우선한다.
- ChatGPT 이미지 생성은 사진, 일러스트, 스티커성 장식, 빈 상태 화면 아트에 우선 사용한다.
- 피드 사진은 너무 광고 같거나 과보정된 결과를 피한다.
- 여운은 사진이 주인공이므로, 장식 일러스트는 피드/상세 본문보다 빈 상태나 온보딩에 집중한다.
